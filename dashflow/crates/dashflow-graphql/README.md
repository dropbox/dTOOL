# dashflow-graphql

GraphQL tool for DashFlow - Execute GraphQL queries and mutations

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-graphql --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-graphql = "1.11"
```
