# dashflow-openweathermap

OpenWeatherMap API integration for DashFlow - Current weather and forecast data

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-openweathermap --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-openweathermap = "1.11"
```
