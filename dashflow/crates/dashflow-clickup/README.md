# dashflow-clickup

ClickUp tools for DashFlow - create tasks, lists, and folders via ClickUp REST API v2

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-clickup --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-clickup = "1.11"
```
