# dashflow-file-tool

File management tools for DashFlow - read, write, list, copy, move, delete, and search files

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-file-tool --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-file-tool = "1.11"
```
