# dashflow-prompts

Prompt management for LLM applications - versioning, registry, performance tracking, and A/B testing.

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-prompts --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-prompts = "1.11"
```
