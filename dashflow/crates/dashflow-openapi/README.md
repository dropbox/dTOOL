# dashflow-openapi

OpenAPI tool for DashFlow - execute API operations from OpenAPI/Swagger specifications

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-openapi --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-openapi = "1.11"
```
