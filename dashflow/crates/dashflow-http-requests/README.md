# dashflow-http-requests

HTTP request tools for DashFlow - GET, POST, PUT, PATCH, DELETE operations

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-http-requests --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-http-requests = "1.11"
```
