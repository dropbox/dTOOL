# dashflow-github

GitHub integration tools for DashFlow - manage issues, PRs, files, and code search

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-github --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-github = "1.11"
```
