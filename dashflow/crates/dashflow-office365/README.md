# dashflow-office365

Office 365 tools for DashFlow - send messages, create events, search emails via Microsoft Graph API

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-office365 --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-office365 = "1.11"
```
