# dashflow-timescale

TimescaleVector (pgvectorscale) integration for dashflow - PostgreSQL with DiskANN and SBQ compression

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-timescale --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-timescale = "1.11"
```
