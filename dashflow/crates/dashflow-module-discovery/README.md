# dashflow-module-discovery

Zero-marker auto-discovery of DashFlow modules from Rust source.

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-module-discovery --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-module-discovery = "1.11"
```
