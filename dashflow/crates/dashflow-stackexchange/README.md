# dashflow-stackexchange

Stack Exchange API integration for DashFlow - search questions, answers, and developer resources

## Documentation

- **API Reference** - Generate with `cargo doc --package dashflow-stackexchange --open`
- **[Main Repository](../../README.md)** - Full project documentation

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
dashflow-stackexchange = "1.11"
```
