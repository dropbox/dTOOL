# Security report

Please do not write anything confidential here, as it will be visible to Gitlab admins.

Follow these steps:

1. Mark this issue as confidential using the checkbox beneath the description field.
2. Set the issue title to **SECURITY ISSUE**.
3. In the body of the issue, describe the security vulnerability with reproduction steps.
4. A maintainer will reach out to you to coordinate disclosure.
