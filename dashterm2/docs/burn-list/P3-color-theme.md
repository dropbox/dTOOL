## Category Notes

This category covers color presets, themes, background images, transparency, cursor colors, and highlighting issues.

### Common Patterns

1. **Background Image Issues** - Scaling, tiling, transparency with background images
2. **Color Preset Problems** - Presets not applying, colors not matching Terminal.app
3. **Dark Mode Sync** - Issues with automatic theme switching based on system appearance
4. **Cursor Color** - Smart cursor color not working correctly in various scenarios
5. **Highlight/Selection** - Text selection highlighting problems with triggers or in specific contexts
6. **256-Color/True Color** - ANSI color rendering differences, terminfo issues
7. **Transparency Effects** - Toggle transparency, background color opacity settings

### Related Files

- `sources/iTermColorPresets.m` - Color preset loading and management
- `sources/iTermColorMap.m` - Color mapping for terminal
- `sources/iTermTextDrawingHelper.m` - Text and color rendering
- `sources/iTermBackgroundColorRenderer.m` - Background color handling
- `sources/iTermImageInfo.m` - Background image handling
- `sources/PTYTextView.m` - Main text view with selection/highlight
