## Category Notes

P3 Other triage **COMPLETE**. All 801 issues categorized. This category covers miscellaneous issues that don't fit other categories, including password manager, triggers, marks, locale, update checker, and general UI issues.

### Common Patterns

1. **Password Manager Integration** - 1Password CLI not found, password entry issues, multi-account support
2. **Trigger Problems** - Triggers firing multiple times, regex not matching all occurrences
3. **Locale/Encoding** - LC_CTYPE errors, Chinese/Unicode display issues, encoding problems
4. **Update Checker** - Updates not working, beta build updates, nightly builds broken
5. **Stage Manager/Multiple Displays** - Visual glitches when using macOS Stage Manager or multiple monitors
6. **Session Restore** - Prior sessions not going away, restored sessions not restoring correctly
7. **Mouse/Cursor Issues** - Cursor flickering, disappearing, or giant cursor problems
8. **Full Screen Issues** - Dock showing in full screen, focus issues with hotkey windows
9. **Memory/Resource Issues** - High memory usage, can't open more than N instances
10. **Build/Install Issues** - Can't build from source, installation path problems
