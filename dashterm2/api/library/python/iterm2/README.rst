DashTerm2 Python API
--------------------

DashTerm2 has a scripting API. It uses Google Protocol Buffers to allow programs written in any language to communicate with DashTerm2. Most users will prefer an interface in a high-level language. This library offers a Python interface to DashTerm2. Please note that Prefs > General > Magic > Enable Python API server must be enabled in DashTerm2 for this library to work.
